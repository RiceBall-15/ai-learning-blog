基于 Astro 构建的技术博客，记录 LangChain4j 框架的学习笔记。

## 📖 博客介绍

这个博客记录了我学习 **LangChain4j** 框架的过程，包括：

- 核心概念解析
- 技术原理解析
- 代码示例和实践
- 最佳实践总结
- 进阶用法探索

## 🛠️ 技术栈

- **Astro** - 现代静态站点生成器
- **GitHub Pages** - 托管服务
- **深色主题** - 护眼的阅读体验

## 🚀 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 构建

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```
